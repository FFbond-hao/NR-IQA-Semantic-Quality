## 1.1 文本语义要素分解的数学模型

### 定义 1.1.1：语义要素集合
将文本提示词 P 分解为四类互补的语义要素：

    P = {S, A, C, St}
    
其中：
- S (Subject): 主体对象 - 名词实体的集合
- A (Attributes): 属性描述 - 形容词/副词的集合  
- C (Context/Scene): 场景关系 - 空间位置关系的集合
- St (Style): 风格指令 - 艺术类型和处理方式的集合

### 数学表达：

令提示词为 P = {w₁, w₂, ..., wₙ}，其中 wᵢ 是第 i 个词语

定义四个词汇集合：
- V_S：主体词汇表 (Subject vocabulary)
- V_A：属性词汇表 (Attribute vocabulary)
- V_C：场景词汇表 (Context vocabulary)
- V_St：风格词汇表 (Style vocabulary)

则语义分解结果为：
```
S = {wᵢ | wᵢ ∈ P ∩ V_S}
A = {wᵢ | wᵢ ∈ P ∩ V_A}
C = {wᵢ | wᵢ ∈ P ∩ V_C}
St = {wᵢ | wᵢ ∈ P ∩ V_St}
```

### 物理意义：

- **主体对象 (S)**：图像中最主要的被描述对象，是图像内容的核心
  - 统计意义：数据的中心位置
  - 物理意义：光学在物体表面的投影

- **属性描述 (A)**：对主体对象的修饰和限定
  - 统计意义：数据的分布特征（如颜色直方图）
  - 物理意义：光学的强度和频谱特性

- **场景关系 (C)**：物体在三维空间中的位置关系
  - 统计意义：图像中物体的几何位置
  - 物理意义：物体的空间投影和透视关系

- **风格指令 (St)**：图像处理或艺术表现方式
  - 统计意义：图像的视觉风格特征
  - 物理意义：光学滤波或后处理的影响

## 2.1 基于 CLIP 的语义相似度

### 定义 2.1.1：CLIP 语义相似度

CLIP（Contrastive Language-Image Pre-training）模型将图像和文本投影到同一
个语义空间中，使用余弦相似度衡量它们的语义相关性。

**数学定义：**
```
sim_CLIP(I, P) = (f_v(I) · f_t(P)) / (||f_v(I)|| · ||f_t(P)||)
```

其中：
- I：输入图像
- P：文本提示词
- f_v：视觉编码器 (vision encoder)，将图像映射到高维特征空间
- f_t：文本编码器 (text encoder)，将文本映射到同一特征空间
- ·：向量点积
- ||·||：向量的欧几里得范数（L2 范数）

**取值范围：**
```
sim_CLIP ∈ [-1, 1]
```

其中：
- sim_CLIP = 1：完全相似（方向相同）
- sim_CLIP = 0：正交（无相关性）
- sim_CLIP = -1：完全相反（方向相反）

**物理意义：**
- 表示生成的图像与文本提示词在语义空间中的相近程度
- 高相似度意味着图像的视觉内容与文本描述高度匹配
- 低相似度表示图像没有正确反映文本意图

**归一化方法：**
为了映射到 [0, 1] 区间，使用线性变换：
```
sim_CLIP_norm = (sim_CLIP + 1) / 2
```

这样：
- sim_CLIP_norm = 1：完全相似
- sim_CLIP_norm = 0.5：无相关性
- sim_CLIP_norm = 0：完全相反

### 定义 2.1.2：关键词匹配率（Keyword Matching Rate）

提取提示词中的关键词，评估这些关键词在图像内容中的出现程度。

**数学定义：**
```
KMR = (Σᵢ match(keywordᵢ, I)) / |keywords|
```

其中：
- keywords：从提示词 P 中提取的关键词集合
- match(keyword, I)：二值函数，若关键词在图像中被检测到则为 1，否则为 0
- |keywords|：关键词总数

**关键词检测方法：**

对于名词关键词（主体），使用对象检测器（如 YOLO、Faster R-CNN）：
```
match(noun, I) = {
    1, if detection_confidence > τ_obj
    0, otherwise
}
其中 τ_obj 是对象检测的置信度阈值（通常 0.5-0.7）
```

对于形容词关键词（属性），使用特征提取：
```
match(adjective, I) = {
    1, if descriptor_similarity > τ_attr
    0, otherwise
}
```

**取值范围：**
```
KMR ∈ [0, 1]
```

**物理意义：**
- 衡量图像中包含的有效信息内容与提示词的匹配程度
- 高 KMR 意味着图像中出现了提示词中提到的大多数关键元素
- 低 KMR 表示图像缺少关键信息或包含不相关的内容

### 定义 2.1.3：综合语义保真度指标

**数学定义：**
```
S_fidelity = α · sim_CLIP_norm + (1-α) · KMR
```

其中：
- α ∈ [0, 1] 是权重系数（默认 α = 0.7）
- sim_CLIP_norm ∈ [0, 1]：CLIP 相似度（已归一化）
- KMR ∈ [0, 1]：关键词匹配率

**参数说明：**
- α = 0.7：给予 CLIP 相似度更高的权重，因为它更全局、更能捕捉整体语义
- (1-α) = 0.3：关键词匹配率作为补充，确保显式提到的对象不被遗漏

**取值范围：**
```
S_fidelity ∈ [0, 1]
```

**统计意义：**
- 这是一个加权平均，结合了两种不同的语义评估方式
- α 的选择基于实验中两个指标对最终质量的贡献程度

**评分解释：**
```
S_fidelity > 0.8：优秀 - 高语义保真度
0.6 < S_fidelity ≤ 0.8：良好 - 语义匹配度好
0.4 < S_fidelity ≤ 0.6：一般 - 语义匹配度中等
0.2 < S_fidelity ≤ 0.4：较差 - 明显的语义偏差
S_fidelity ≤ 0.2：很差 - 严重偏离提示词意图
```

## 3.1 清晰度指标（Sharpness）

### 定义 3.1.1：基于拉普拉斯算子的清晰度

拉普拉斯算子是一个二阶微分算子，用于检测图像中的边缘和细节。清晰图像在
应用拉普拉斯算子后会产生较大的响应值。

**数学定义：**

拉普拉斯算子：
```
∇²I = ∂²I/∂x² + ∂²I/∂y²
```

离散形式的拉普拉斯算子（3×3 卷积核）：
```
L = [  0  -1   0 ]
    [ -1   4  -1 ]
    [  0  -1   0 ]
```

清晰度分数定义为：
```
Sharpness_score = Var(∇²I)
```

其中 Var 表示方差。

**理论推导：**

1. 对灰度图像 I 应用拉普拉斯卷积
2. 计算卷积结果的方差

```
Sharpness = (1/HW) Σᵢ₌₁ʰ Σⱼ₌₁ʷ (Lᵢⱼ - μ_L)²

其中：
- H, W：图像的高度和宽度
- Lᵢⱼ = ∇²I(i,j)：图像 (i,j) 处的拉普拉斯值
- μ_L = (1/HW) ΣΣ Lᵢⱼ：所有拉普拉斯值的均值
```

**规范化方法：**

使用 Sigmoid 函数进行非线性映射：
```
Sharpness_norm = 1 / (1 + e^(-Sharpness_score/τ_s))

其中 τ_s 是温度参数（通常设为 1000）
```

结果范围 Sharpness_norm ∈ [0, 1]

**物理意义：**

- **拉普拉斯值的方差大**：图像包含丰富的高频分量，对应清晰的边缘和细节
- **拉普拉斯值的方差小**：图像主要是平坦区域，对应模糊图像
- **方差为 0**：完全均匀的图像（完全模糊）

**统计意义：**

- 清晰度是**局部对比度**的一个度量
- 方差直接反映了图像的**信息熵**程度
- 高清晰度对应高信息熵

**评分解释：**
```
Sharpness > 0.7：清晰，细节丰富
0.5 < Sharpness ≤ 0.7：较清晰，细节可见
0.3 < Sharpness ≤ 0.5：一般清晰度
Sharpness ≤ 0.3：模糊，细节丧失
```


## 3.2 噪声指标（Noise Level）

### 定义 3.2.1：基于频域分析的噪声

噪声通常表现为随机的、高频的波动。通过频域分析（傅里叶变换），可以评估
图像中的噪声水平。

**数学定义：**

对图像进行二维傅里叶变换：
```
F(u,v) = FFT2D(I)
```

计算幅度谱：
```
|F(u,v)| = √(Real(F)² + Imag(F)²)
```

噪声指标定义为高频分量占总能量的比例：
```
Noise = E_high / E_total

其中：
- E_total = Σᵤ Σᵥ |F(u,v)|²：图像的总能量
- E_high = Σ_{(u,v) ∈ high_freq} |F(u,v)|²：高频能量
```

**高频区域定义：**

在傅里叶频域中心化后，高频被定义为距离频域中心距离超过阈值的区域：
```
high_freq = {(u,v) | dist((u,v), center) > r_threshold}

其中：
- dist：欧几里得距离
- r_threshold = 0.3 × max_distance：阈值通常设为最大距离的 30%
```

**数学公式：**
```
Noise = (Σ_{(u,v) ∈ high_freq} |F(u,v)|²) / (Σᵤ Σᵥ |F(u,v)|²)
```

**取值范围：**
```
Noise ∈ [0, 1]
- Noise ≈ 0：低噪声，频域集中在低频分量
- Noise ≈ 1：高噪声，高频能量占主导
```

**物理意义：**

- **低噪声（Noise < 0.3）**：图像主要由低频分量组成，表示图像平滑
- **中等噪声（0.3 < Noise < 0.6）**：含有合理的高频细节
- **高噪声（Noise > 0.6）**：高频噪声占主导，图像质量下降

**统计意义：**

- 噪声是**信号的无用部分**
- 用频率分量的比例来量化噪声的强度
- 高噪声对应低的**信噪比（SNR）**

**Parseval 定理的应用：**

根据 Parseval 定理，时域能量等于频域能量：
```
Σᵢⱼ |I(i,j)|² = Σᵤᵥ |F(u,v)|²
```

这保证了我们的频域分析在物理上是等价的。


## 3.3 伪影指标（Artifacts）

### 定义 3.3.1：边缘连续性分析

伪影通常表现为不自然的边缘断裂。通过分析梯度的局部变化，可以检测伪影。

**数学定义：**

计算 Sobel 梯度：
```
Gₓ = Sobel_x * I
Gᵧ = Sobel_y * I

其中 Sobel_x 和 Sobel_y 是 Sobel 算子
```

梯度幅度：
```
G = √(Gₓ² + Gᵧ²)
```

局部方差（用 5×5 窗口）：
```
Var_local(i,j) = Var(G[i-2:i+2, j-2:j+2])
```

边缘连续性指标：
```
EdgeDiscontinuity = Std(Var_local)

其中 Std 是标准差
```

**物理意义：**

- **高的边缘不连续性**：局部梯度变化不规则，表示有伪影
- **低的边缘不连续性**：梯度变化平滑，边缘质量好

### 定义 3.3.2：频域异常检测

某些伪影（如 JPEG 块状伪影）在频域中表现为异常的峰值。

**数学定义：**

计算频谱峰值与背景的比值：
```
FrequencyAnomaly = E_edges / E_center

其中：
- E_center：频域中心区域的能量
- E_edges：频域边缘区域的能量
```

高比值表示存在块状伪影。

### 定义 3.3.3：综合伪影指标

**数学定义：**
```
Artifacts = σ(EdgeDiscontinuity + FrequencyAnomaly) / 2

其中 σ 是 Sigmoid 函数：
σ(x) = 1 / (1 + e^(-x))
```

**取值范围：**
```
Artifacts ∈ [0, 1]
- Artifacts ≈ 0：无伪影
- Artifacts ≈ 1：严重伪影
```


## 3.4 综合技术质量指标（Combined Technical Quality）

### 定义 3.4.1：加权融合

**数学定义：**
```
Q_technical = β₁·Sharpness + β₂·(1-Noise) + β₃·(1-Artifacts)

其中：
- β₁ = 0.4：清晰度权重
- β₂ = 0.3：噪声权重（取反，因为噪声越低越好）
- β₃ = 0.3：伪影权重（取反，因为伪影越少越好）
- β₁ + β₂ + β₃ = 1
```

**权重设置的依据：**

1. **清晰度 (40%)**：对图像质量的影响最大
   - 是最直观的质量指标
   - 与视觉感知的相关性最高

2. **噪声 (30%)**：严重影响图像质量
   - 噪声使细节难以辨认
   - 但略低于清晰度的重要性

3. **伪影 (30%)**：同等于噪声的影响
   - 块状伪影等会明显降低质量
   - 但人眼不如对清晰度敏感

**取值范围：**
```
Q_technical ∈ [0, 1]
```

**物理意义：**

这个指标综合反映了图像从**图像处理角度**的质量：
- 是否包含细节（清晰度）
- 是否有随机干扰（噪声）
- 是否有处理伪影（伪影）

## 4.1 边缘连续性指标（Edge Continuity）

### 定义 4.1.1：数学定义

通过形态学操作和连通分量分析，评估对象边界的连贯性。

**步骤 1：边缘提取**
```
edges = Sobel(I) > threshold
```

**步骤 2：形态学闭操作**
```
edges_closed = binary_closing(edges, iterations=1)

闭操作 = 膨胀 + 腐蚀，用于填充小孔洞
```

**步骤 3：连通分量标记**
```
labeled, num_components = label(edges_closed)
```

**步骤 4：计算连续性**
```
连续性 = (最大连通分量的大小) / (所有边缘像素总数)

E_continuity = max(size(component_i)) / total_edge_pixels
```

**物理意义：**

- **高连续性（>0.7）**：对象的边界形成连贯的轮廓，结构清晰
- **低连续性（<0.3）**：对象边界支离破碎，可能是质量问题

**统计意义：**

- 这是一个**连通性比例**的度量
- 高比例意味着大部分边缘像素属于同一对象


## 4.2 形状规则性指标（Shape Regularity）

### 定义 4.2.1：Polsby-Popper 圆形度指标

用来衡量对象形状与圆形的接近程度。

**数学定义：**
```
圆形度 = 周长² / (4π × 面积)

CircularityIndex = Perimeter² / (4π × Area)
```

**理论推导：**

对于完美圆形：
```
周长 = 2πr
面积 = πr²

圆形度 = (2πr)² / (4π × πr²) = 4π²r² / (4π²r²) = 1
```

对于其他形状，圆形度会大于 1。

**规范化：**

为了映射到 [0, 1]，使用：
```
Regularity = 1 / (1 + CircularityIndex)
```

这样：
- 完美圆形：Regularity = 0.5
- 正方形：Regularity ≈ 0.45
- 不规则形状：Regularity < 0.3

**物理意义：**

- **高规则性**：对象形状规则，边界平滑
- **低规则性**：对象形状不规则，可能表示质量问题或伪影

**统计意义：**

- 这是一个**形状复杂度**的倒数
- 高规则性对应低复杂度


## 4.3 对象完整性指标（Object Completeness）

### 定义 4.3.1：孔洞检测

检测对象内部是否存在不应有的孔洞，表示对象的缺失或损坏。

**数学定义：**

使用二值化和填充操作：
```
binary = (I > threshold)
filled = binary_fill_holes(binary)

孔洞面积 = sum(filled) - sum(binary)
对象面积 = sum(binary)

Hole_ratio = 孔洞面积 / 对象面积
```

完整性指标：
```
Completeness = {
    1, if Hole_ratio < 0.1
    0, if Hole_ratio ≥ 0.1
}
```

对所有对象求平均：
```
Object_Completeness = (完整对象数) / (总对象数)
```

**物理意义：**

- **高完整性**：对象形状完整，没有缺失
- **低完整性**：对象存在明显孔洞或破损

**阈值设定的理由：**

- 10% 阈值基于人眼可感知的缺失程度
- 小于 10% 的孔洞可能是自然的纹理特征


## 4.4 综合结构完整性指标（Combined Structural Integrity）

### 定义 4.4.1：加权融合

**数学定义：**
```
I_structural = γ₁·E_continuity + γ₂·Regularity + γ₃·Completeness + γ₄·Smoothness

其中：
- γ₁ = 0.3：边缘连续性权重
- γ₂ = 0.25：形状规则性权重
- γ₃ = 0.25：对象完整性权重
- γ₄ = 0.2：边界平滑度权重
- Σγᵢ = 1
```

**权重设置的依据：**

1. **边缘连续性 (30%)**：最直接反映结构质量
2. **形状规则性 (25%)**：表示形状的自然性
3. **对象完整性 (25%)**：同等重要
4. **边界平滑度 (20%)**：辅助指标

## 5.1 综合质量指数

### 定义 5.1.1：三维度加权融合

**数学定义：**
```
Q_final = w₁·S_fidelity + w₂·Q_technical + w₃·I_structural

其中：
- w₁ = 0.35：语义保真度权重
- w₂ = 0.40：技术质量权重
- w₃ = 0.25：结构完整性权重
- w₁ + w₂ + w₃ = 1
```

**取值范围：**
```
Q_final ∈ [0, 1]
```

### 定义 5.1.2：质量等级映射

**数学定义：**
```
Level(Q_final) = {
    "优秀",    if Q_final > 0.80
    "良好",    if 0.60 < Q_final ≤ 0.80
    "中等",    if 0.40 < Q_final ≤ 0.60
    "较差",    if 0.20 < Q_final ≤ 0.40
    "很差",    if Q_final ≤ 0.20
}
```

### 定义 5.1.3：权重参数的统计意义

**权重选择的依据分析：**

| 维度 | 权重 | 理由 |
|------|------|------|
| 语义保真度 | 0.35 | 对文本驱动生成最重要；决定是否生成了正确的内容 |
| 技术质量 | 0.40 | 最高权重；视觉质量是所有应用的基础 |
| 结构完整性 | 0.25 | 较低权重；在高技术质量下，通常自动满足 |

**理论论证：**

1. **为什么技术质量最重要？**
